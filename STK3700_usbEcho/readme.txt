USB  CDC ECHO EXAMPLE for EFM32 microcontrollers

This example project uses the EFM32 USB Device protocol stack
==========================
Implements an USB CDC based virtual COM port. 
When You send a data packet from a host site (PC) device appears
like an echo module - immediately responds the same data to the host


Board:  Silicon Labs EFM32GG-STK3700 Development Kit
Device: EFM32GG990F1024
